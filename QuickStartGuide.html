About Next.js React Apps

A Next.js React app is a web application built using React, a JavaScript library for creating interactive user interfaces, enhanced by Next.js, a framework that simplifies building fast, scalable websites and apps.

Prerequisites

  • Node.js 16.x or higher - Download from nodejs.org
  • npm 7.x or higher - Included with Node.js
  • PowerShell or Command Prompt - Built into Windows (search "PowerShell" in Start menu)

Development Mode

  1. Extract Singularity.zip to any folder you prefer.
  2. Open PowerShell/Command Prompt and navigate to the project: 
    cd "path\to\your\Singularity"
  3. Install dependencies: 
    npm install
  4. Start the development server: 
    npm run dev
  5. Visit http://localhost:3000 in your browser.

Production Mode

  1. Extract Singularity.zip to any folder you prefer.
  2. Open PowerShell/Command Prompt and navigate to the project: 
    cd "path\to\your\Singularity"
  3. Install dependencies (if not already done): 
    npm install
  4. Build the project: 
    npm run build
  5. Start the production server: 
    npm start
  6. Visit http://localhost:3000 in your browser.

Troubleshooting Dependencies

If you encounter any issues with packages:

  1. Delete the node_modules folder (if present)
  2. Run: 
    npm install
  3. Then follow the start instructions above

Note: Ensure fonts in public/assets/fonts/ibm-plex-sans/ are present. Use Chrome, Firefox, Safari, or Edge for best compatibility.
